# HPC Summer School

A collection of exercises I worked on while attending the HPC Summer School held by EPCC (Edinburgh Parallel Computing Centre), run on the ARCHER2 national supercomputer.

The exercises cover:

- **Parallel programming** using OpenMP and Open MPI, across both CPUs and GPUs
- **Machine learning** fundamentals and applications in an HPC context
